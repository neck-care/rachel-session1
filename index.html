Rachel · 第一节线上课程

评估 · 松解 · 训练（首节课 55-60 分钟）
评估(15-20min)松解(~10min)训练(~30min)

Step 1 · 线上评估

请准备好：瑜伽垫/床、一面墙、手机计时器

1

颈椎旋转活动度

  1. 坐姿，缓慢向左转头至受限停
  2. 缓慢向右转头至受限停
  3. 对比左右旋转幅度
2

头后仰症状

  1. 坐姿，头微微后仰
  2. 感受是否有眩晕、不稳感
3

颈部肌力

  1. 平躺，收下巴，脖子略抬起离开床面
  2. 保持此姿势，记录坚持秒数
4

肩胛稳定性 — 靠墙天使

  1. 靠墙站立，后脑勺、肩胛、臀、脚跟贴墙
  2. 双臂做W形→Y形上滑（天使翅膀动作）
  3. 观察肩胛骨是否离开墙面
5

核心控制 — 死虫式

  1. 仰卧，抬起四肢呈桌面姿势
  2. 对侧手脚同时缓慢下放（右手+左脚下放）
  3. 观察腰部是否紧贴地面
6

体态观察

  1. 侧面站立，自然放松
  2. 观察耳垂与肩峰连线（铅垂线参考）
  3. 平躺观察肩胛骨是否贴合地面
7

前庭 — 单腿站立

  1. 闭眼，双手叉腰，单腿站立
  2. 先左腿，后右腿，各一次
  3. 记录坚持时间与摇晃程度
8

前庭 — 转圈

  1. 闭眼，原地缓慢转一圈
  2. 先向左转，后向右转，各一次
  3. 感受转后眩晕程度
9

症状触发

  1. 快速从坐到站，重复3次
  2. 观察是否诱发头晕或不稳感

Step 2 · 松解

1. 枕下肌群筋膜球松解（2分钟） 枕下肌群

  1. 平躺，双联筋膜球卡在枕骨下缘凹陷处
  2. 闭眼放松，下巴微收
  3. 做毫米级微小点头动作，利用头部自重压揉
02:00
作用：头晕核心原因——松解枕下肌群降低上颈椎关节张力

2. 胸锁乳突肌筋膜球松解（2分钟） 胸锁乳突肌

  1. 侧卧，筋膜球放在耳垂下方胸锁乳突肌位置
  2. 缓慢上下移动寻找压痛点
  3. 找到压痛点维持20-30秒
  4. 左右各1分钟
02:00
作用：头前倾+旋转受限核心因素

3. 胸小肌筋膜球松解（2分钟） 胸小肌

  1. 靠墙站立，筋膜球放在锁骨下方胸小肌位置
  2. 身体缓慢上下移动寻找压痛点
  3. 找到压痛点维持20-30秒
  4. 左右各1分钟
02:00
作用：改善头前倾和肩膀回缩

4. 门框胸大肌拉伸（2分钟） 胸大肌

  1. 站门框旁，前臂贴门框，肘与肩同高
  2. 身体微前倾感受拉伸
  3. 保持躯干正直不耸肩
  4. 维持30秒x4次
02:00
作用：打开被压缩的胸廓

5. 斜方肌上束拉伸（2分钟） 斜方肌上束

  1. 坐姿，头向右侧偏（右耳靠近右肩），右手轻压左侧头部
  2. 感受左侧斜方肌拉伸
  3. 维持30秒，换侧
  4. 左右各2次
02:00
作用：缓解左侧旋转受限

Step 3 · 训练

1. 9090呼吸（3分钟）交感神经

  1. 仰卧双腿屈膝90度踩墙
  2. 鼻吸气4秒让肚子鼓起来，嘴吐气8秒，屏息3秒
  3. 重复8-10次
03:00
作用：激活副交感神经，降低交感张力，为后续训练打底

2. 平躺吸舌收下巴（3分钟）深层稳定

  1. 仰卧，后脑勺垫薄书
  2. 舌顶上颚，下巴平行后收2毫米
  3. 感受后脑勺与书本轻压
  4. 保持10秒，重复10次
03:00
作用：激活颈深屈肌（DNF），重建颈椎中立位

3. 平躺等长抗阻-后侧（3分钟）深层稳定

  1. 仰卧，双手十指交叉托后脑勺
  2. 头向后发力，双手向前抵抗（5%力量）
  3. 头部不产生位移，保持6秒
  4. 重复8次
03:00
作用：通过神经自体抑制机制，降低颈部深层痉挛

4. 平躺双指眼球追踪（3分钟）前庭

  1. 仰卧，双臂伸直举大拇指，与肩同宽
  2. 头颅保持绝对静止，眼球平滑从左拇指横移到右拇指（4秒）
  3. 定格1秒后4秒移回
  4. 8次/组，3组
03:00
作用：躺姿安全训练前庭眼反射，不触发眩晕

5. 眼神死锁微动摇点脑（3分钟）前庭

  1. 仰卧，左手拇指竖在眼前30cm
  2. 双眼死盯拇指，头面极慢右转2度→左转2度
  3. 视线始终不离开目标
  4. 2组x10次
03:00
作用：前庭-颈椎联合训练，提升视觉稳定性

6. 四足跪姿胸椎转体（3分钟）灵活性

  1. 四点跪姿，一手扣后脑勺
  2. 呼气肘向下向内，吸气胸椎旋转打开
  3. 眼睛盯肘尖
  4. 双侧各10次
03:00
作用：恢复胸椎旋转活动度，解除颈椎代偿

7. 仰卧位极端眼动屏息抑制（4分钟）前庭/枕下肌

  1. 仰卧平躺，屈膝踩地，后脑勺垫薄书
  2. 头部保持绝对静止
  3. 双眼极限看右下角，鼻吸气4秒，屏息3秒
  4. 嘴长呼气8秒，眼球缓慢回正
  5. 换左下角重复，左右各5次
04:00
作用：眼-颈反射抑制：极限眼动+屏息+呼气诱发枕下肌群自发性放松（PIR效应），不触发眩晕的前提下下调上颈椎张力

8. 仰卧位微点头联动平滑追踪（4分钟）前庭/DNF

  1. 仰卧，手持卡片（或手指）置于眼前40cm
  2. 舌顶上颚，做极小收下巴（2毫米，胸锁乳突肌不发硬）
  3. 保持收下巴力量，眼球平滑追踪卡片左右移动（各15度）
  4. 连续追踪往返5次/组，3组
04:00
作用：颈深屈肌稳定+眼球平滑追踪联合训练，重建大脑多感知通道整合，建立抗晕"护城河"
